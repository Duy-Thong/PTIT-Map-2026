## ADDED Requirements

### Requirement: New Parking Hotspot
A new interactive hotspot for "Gửi xe" must be added to the map.

#### Scenario: Parking spot visibility
- **WHEN** the map is loaded
- **THEN** a marker icon should appear at the lower-left area (near the designated "nhà xe" area)

### Requirement: Parking Information
The parking spot must provide instructions when clicked.

#### Scenario: User clicks parking marker
- **WHEN** the user clicks the "Gửi xe" marker on the map
- **THEN** a popup appears with text: "Khu vực gửi xe dành cho khách và sinh viên dự lễ. Vui lòng xếp xe gọn gàng theo hướng dẫn của bảo vệ."
